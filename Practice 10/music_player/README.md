# 🎵 Music Player

An interactive music player with keyboard controls for playing audio files.

## ✨ Features

- Play/Stop/Next/Previous track controls
- Playlist management (MP3 and WAV support)
- Display current track information
- Real-time playback position tracking
- Clean text-based UI

## 🎮 Controls

| Key | Action |
|-----|--------|
| **P** | Play current track |
| **S** | Stop playback |
| **N** | Next track |
| **B** | Previous track |
| **Q** | Quit application |

## 🚀 How to Run

```bash
cd music_player
python main.py
