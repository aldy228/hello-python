# Practice 10: Pygame Projects

Three complete games demonstrating Pygame fundamentals: racing, snake, and painting.

## 🎮 Projects

### 1. 🏎️ Racer
Car racing game with coin collection and level progression.
- Avoid enemy cars
- Collect coins for bonus points
- Level up to increase difficulty

### 2. 🐍 Snake
Classic snake with modern features.
- Wall and self-collision detection
- Smart food spawning
- Level-based speed increases

### 3. 🎨 Paint
Feature-rich drawing application.
- Multiple drawing tools (brush, rect, circle)
- Eraser and color selection
- Adjustable brush sizes

## 📋 Requirements
pygame>=2.6.0

## 🚀 Installation & Run
```bash
# Install dependencies
pip install -r requirements.txt

# Run any game:
cd racer && python main.py
cd ../snake && python main.py
cd ../paint && python main.py
