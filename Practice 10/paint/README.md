# 🎨 Paint

A simple drawing application with multiple tools.

## ✨ Features
- **Brush**: Freehand drawing with adjustable size
- **Rectangle**: Click and drag to draw rectangles
- **Circle**: Click and drag to draw circles
- **Eraser**: Remove drawings (white color)
- **Color Selection**: 8 preset colors in toolbar
- **Brush Size**: 4 size options (3, 5, 10, 20px)
- **Save**: Press S to save drawing as PNG

## 🎮 Controls
### Mouse
- Click + Drag on canvas: Draw
- Click toolbar: Select color/tool/size

### Keyboard Shortcuts
| Key | Action |
|-----|--------|
| B | Select Brush |
| R | Select Rectangle |
| C | Select Circle |
| E | Select Eraser |
| S | Save drawing |

## 📂 Files
- `main.py` - Main loop, event handling, UI
- `tools.py` - DrawingTools class for all drawing operations
