# ⏰ Mickey's Clock

An analog clock application using Mickey Mouse hands to display the current time.

## ✨ Features

- Real-time clock display (minutes and seconds)
- Right hand = Minutes
- Left hand = Seconds
- Smooth rotation synchronized with system time
- Clean visual design with clock face

## 🎮 How to Run

```bash
cd mickeys_clock
python main.py
