# 🔴 Moving Ball Game

A simple interactive game demonstrating basic movement and boundary detection.

## ✨ Features

- Red ball (50px diameter) on white background
- Smooth arrow key controls
- Boundary checking (ball cannot leave screen)
- 20-pixel movement step per key press
- 60 FPS smooth animation

## 🎮 Controls

| Key | Action |
|-----|--------|
| **↑** | Move Up |
| **↓** | Move Down |
| **←** | Move Left |
| **→** | Move Right |

## 🚀 How to Run

```bash
cd moving_ball
python main.py
