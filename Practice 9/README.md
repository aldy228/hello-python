# Practice 10: Game Development with Pygame

This project contains three classic games built with Pygame to demonstrate graphics rendering, input handling, and animations.

## 🎮 Games Included

1. **Mickey's Clock** - Analog clock with rotating Mickey Mouse hands
2. **Music Player** - Keyboard-controlled audio player with playlist support
3. **Moving Ball** - Interactive ball movement game with boundary checking

## 📋 Requirements

- Python 3.11+
- Pygame 2.6.1

## 🚀 Installation

pip install -r requirements.txt
